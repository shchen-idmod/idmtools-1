===================
Supported platforms
===================

|IT_l| currently supports running on the following platform:

**Local**: You can run simulations and analysis locally on your computer within docker container, rather than on a remote high-performance computer (HPC). For more information about these modules, see :doc:`../api/idmtools_platform_local_index`.

For additional information about configuring idmtools.ini, see :doc:`../configuration`.
