=================
Welcome to |IT_l|
=================

|IT_l| is the idmtools local runner which allows execution of tasks in a local docker container and provides a platform that is somewhat similar to COMPS, though much more limited.


Workflows for using |IT_l| is user-dependent, and can include any of the tasks listed below.
